## Uso de Inteligência Artificial

**Ferramentas utilizadas:** V0 e Claude Code

**Em que etapas a IA ajudou mais?**
A IA foi essencial na criação da estrutura inicial do projeto, geração de componentes e orientação na configuração do ambiente. Também auxiliou na resolução de erros, uso do Git e preparação para deploy.

**O que foi ajustado manualmente e por quê?**
Foram feitos ajustes manuais em estilos, decisões técnicas, organização das pastas e revisão de conflitos de código para garantir clareza, funcionamento correto e melhor adequação aos objetivos do projeto. Algumas decisões foram tomadas manualmente para manter maior controle sobre a aplicação.
